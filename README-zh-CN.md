## 安装
- 如果您的电脑在使用 Windows，那么建议切换至 [GNU/Linux 系统](https://www.gnu.org/distros/free-distros.html) 。如果您不愿意，请继续：
  - （可选）使您的系统可自动登录。
  - 在 Windows 环境下，安装 *Python3* 和一个非 IE 的独立*浏览器*。
  - 下载一份此软件的发行，并解压至某处。
  - 为其中的 *main.py* 创建一个*快捷方式*：右键拖动文件，选择“新建快捷方式”。
  - 将 main.py 的快捷方式与浏览器的快捷方式（通常在桌面）放入 `开始菜单 -> 启动` 文件夹。
  - 重启您的电脑，现在它会自动运行此应用。
- 如果它在运行 GNU/Linux， 那么：
  - （可选）使您的系统可自动登录。
  - 确认包 *python3* 与 *firefox* 已被安装。
  - 下载一份此软件的发行，并解压至某处。且记住其中 main.py 文件的位置。
  - 编辑文件 `(用户主目录)/.bashrc` （或其他与自启动相对应的文件），在文件末尾添加一行：
    - `python3 ~/main.py文件的位置/main.py`
  - 重启您的电脑，现在它会自动运行此应用。

## 小技巧
- 条码与价格系统：
  - 您安装此应用后的第一件事，是*录入商品*至数据库。推荐这样做：
    - 在您的电脑上，打开此应用，确认它已连接到一个网络，并获取它的 IP 地址。
    - 打开您的手机，在无线网络设置中连接与电脑网络处在同一环境的 Wifi。
    - 通过 *OTG 连接器* 连接扫码器至手机。
    - 在手机浏览器，打开网址 `http://电脑的IP地址:8101/`，并打开网页中的菜单。
    - 向下滚动，在“管理”小节，您可以录入商品。只需在条码框扫入条码，输入商品的名称、单位、价格，点击录入按钮。
    - 带着手机和一名帮手，这样在商场转一圈，扫描、录入您所有的商品。
  - 在您拥有自己的数据库后，将数据库导出、导入（在菜单）至其他电脑。
  - 如果有商品未录入，也可在结算时直接输入其*价格*。
  - 在扫描顾客的所有商品后，点击*结算*。浏览器会弹出打印对话框，可以打印账单。

- 条码与会员系统：
  - 录入的条码输入以减号（-）开头时，它被视为*命令*。后面的字母表示操作类型，然后是具体内容，以减号分隔……
  - “-a”是*注册会员*命令，以 `-a10010001000-Name` 的语法可以注册名为 `Name`、ID 为 `10010001000` 的会员。ID 可以是手机号码或其他事物。
  - 注册会员后，会提供一份*会员码*。顾客只需用手机拍摄（非扫描）它。
  - 在用扫码器扫描会员码后，此应用会将当前会话视为*会员会话*，直到结算。
  - 在会员会话，您可以进行一次*使用积分*操作。每次结算后都会给予会员一定的积分。
  - 依靠积分系统，常来的顾客可以兑换一些礼品。这样可以为店铺实现宣传效应，保证长久发展。
